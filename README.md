# Java Calculator

A simple command-line calculator written in plain Java (no build tools needed).

## Features
- Add, subtract, multiply, divide
- Input validation and error handling
- Basic self-written tests (no external libraries)

## How to run

Compile everything:
```bash
javac -d out src/*.java
```

Run the calculator:
```bash
java -cp out Main
```

Run the tests:
```bash
java -cp out CalculatorTest
```
